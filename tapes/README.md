
Each tape image has two versions.  The `.ptp` file is the binary paper
tape image ready to load into the machine.  The `.txt` file is the
ASCII text version of the tape.

* `Character_Set` - Full character set for testing the Flexowriter.
* `Program_Input_Routine_v2_Flexowriter` - PIR image to bootstrap from the
Flexowriter.
* `Program_Input_Routine_v2_Tape_Reader` - PIR image to bootstrap from the
tape reader.
